## Format guidance (for the receiving engine)

- **Preferred:** SVG (vector, scales cleanly in the carousel)
- **Acceptable:** PNG with transparent background, ≥ 1200px on the long edge
- **Avoid:** JPG with white background (visible card on dark sections)

If only horizontal lockups are provided, the carousel card should center them with generous padding rather than stretching. If only square marks are provided, pair them with the project name set in the site's display typeface.

Alternates can live in `_alternates/` next to this README and will be ignored by the build.
